# 🎬 모바일 버전 (9:16, 약 60초)

## 녹화 가이드

### iPhone
1. **설정** → **제어 센터** → **화면 기록** 추가
2. Safari로 https://meet-at-suzak.vercel.app/ 접속
3. 주소창 숨기기: 화면 살짝 위로 스와이프하면 자동으로 사라짐
4. 제어센터 → 화면 기록 빨간 동그라미 길게 → 마이크 끄기 → 녹화 시작

6. 끝나면 상단 빨간 시계 아이콘 탭 → 중지

### Android
1. 퀵 설정 펼치기 → **화면 녹화** 탭
2. 소리 끄기 / 터치 표시 ON
3. Chrome으로 https://meet-at-suzak.vercel.app/ 접속
4. 녹화 시작 → 시나리오 진행

# ✂️ CapCut 편집 가이드 (모바일·데스크탑 공통)

## 1. 영상 불러오기
1. CapCut 열기 → **새 프로젝트**
2. 녹화한 영상 파일 선택 → 가져오기

## 2. 비율 설정
- 모바일 버전: 우측 상단 비율 → **9:16**
- 데스크탑 버전: **16:9**
- 자동으로 안 맞으면 영상 클립 우클릭 → 비율 조정

## 3. 컷 편집 (긴 부분 잘라내기)
- 타임라인에서 클립 선택 → 분할 도구로 자르기
- 너무 느리게 스크롤한 부분 가속(1.5x) 또는 컷
- 총 길이를 모바일 60초 / 데스크탑 75초 안에 맞추기

## 4. 자막 얹기 (가장 중요)
1. 좌측 도구 → **텍스트** → **텍스트 추가**
2. 위 시나리오의 자막 시간에 맞춰 입력
3. 폰트: **Noto Sans KR** 또는 **G마켓 산스** 류 (가독성 ↑)
4. 크기: 모바일은 큼지막하게 (28~36pt), 데스크탑은 22~28pt
5. 색상: 흰색 + 그림자(또는 검은 박스 배경) → 모든 화면에서 가독성 확보
6. 위치: 모바일 = 상단 1/3 또는 하단 1/4 (안전 영역), 데스크탑 = 하단 가운데

## 5. BGM 얹기
1. 좌측 도구 → **오디오** → **음악**
2. CapCut 내장 음원에서 **상업 안전(SAFE)** 표시된 곡 선택
3. 추천 분위기:
   - "Lofi", "Cafe Jazz", "Acoustic", "Chillhop" 검색
   - 추천 곡 1: **"Coffee Shop"** 류 lofi
   - 추천 곡 2: 따뜻한 어쿠스틱 기타
4. 볼륨: 자막보다 묻히지 않게 30~50%
5. 끝 부분 자동 페이드아웃 ON

## 6. 화면 효과 (선택)
- 클립 사이에 **부드러운 페이드 전환** (Dissolve 0.3초)
- 핵심 기능 보여줄 때 **줌 인** 살짝 (10% 정도)
- 너무 많이 쓰면 산만함 — 3~4곳 정도만

## 7. 내보내기
1. 우측 상단 **내보내기** (Export)
2. 해상도: **1080p** / 프레임률: **30fps**
3. 워터마크 제거 옵션 있으면 ON (CapCut 워터마크는 광고처럼 보임)
4. 저장 위치: `c:\Users\User\Desktop\혜빈\영상\`
5. 파일명:
   - 모바일: `카페수작_데모_모바일_9x16.mp4`
   - 데스크탑: `카페수작_데모_데스크탑_16x9.mp4`

5. 추천 검색어: `coffee`, `cafe`, `morning`, `acoustic`

- https://pixabay.com/music/

데스크탑 버전(16:9) MP4 파일을 보고서 슬라이드에 삽입:

1. PowerPoint에서 **삽입** 탭 → **비디오** → **이 디바이스**
2. `카페수작_데모_데스크탑_16x9.mp4` 선택
3. 슬라이드에 영상 추가됨 → 크기·위치 조정
4. 옵션 — **자동 실행** 설정하면 슬라이드 진입 시 자동 재생

# ⚠️ 자주 발생하는 문제

**Q. CapCut 워터마크가 끝에 1초 박혀요**
A. 영상 끝 1초 자르면 제거됨 또는 우측 상단 워터마크 토글 OFF

**Q. 자막 폰트가 한글이 깨져요**
A. CapCut 폰트 → "Korean" 카테고리 → Noto Sans KR 선택

**Q. 화면 녹화 파일이 너무 커요 (1GB 이상)**
A. CapCut에서 한 번 내보내면 자동으로 압축됨. 또는 해상도 720p로 낮추기

**Q. iPhone 녹화 시 사이트 색이 이상해요**
A. Safari보다 Chrome iOS 앱이 더 정확. 또는 다크모드 OFF 확인

**Q. 어디까지 영상 길이 늘려도 되나요?**
A. 쇼츠/릴스는 **60초 이내 권장** (그 이상이면 노출 알고리즘에서 불리). 일반 유튜브는 **75~90초 적정**.
