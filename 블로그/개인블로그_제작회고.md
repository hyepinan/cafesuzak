# 학생이 직접 만든 동네 카페 홈페이지 — 비교과 프로그램 제작 회고

조치원에 있는 작은 카페 '카페수작'의 홈페이지와 매칭 서비스를 비교과 프로그램으로 직접 만들었어요. 디자인 · 개발 · 배포 · 콘텐츠 자산화까지. 약 2주 동안의 작업 과정과 결과물을 짧게 정리합니다.

> **🔗 결과물 보기 · https://meet-at-suzak.vercel.app/match**

---

## 시작 · 왜 만들었나

고려대학교 세종캠퍼스 글로벌경영학과에 다니고 있어요. 이번 학기 학교 비교과 프로그램 — **"지역 영세기업 AI 콘텐츠 마케팅 산학협력"** 과제로, 학교 근처 카페 '카페수작'을 대상 기업으로 받았습니다.

카페수작은 사장님이 매일 매장에서 직접 원두를 로스팅하고, 시럽 · 소스 · 디저트까지 모두 손으로 만드는 작은 로스터리 카페예요. 학교에서 도보 10분. 카공러 사이에 입소문이 난 곳인데, 의외로 디지털 채널 — 인스타 / 검색 결과 / 메뉴 안내 — 에서는 노출이 적었어요.

> "맛은 진짜 좋은데, 매장 밖에서 카페에 대한 정보를 찾기가 어려워요."

이 한 문장에서 출발했어요.

> "그러면 — 한 번 가본 사람이 다시 찾기 좋고, 안 가본 사람도 한 번 가고 싶어지는 페이지를 만들어볼게요."

단순한 영업 정보 안내를 넘어서, 이 카페에서 일어나는 작은 인연까지 담아낼 수 있는 페이지로.

## 무엇을 만들었나

### 1) 메인 홈페이지

- 조치원 + 사용자 위치 실시간 날씨 (Open-Meteo API 연동)
- 기분 10가지 × 메뉴 추천 매칭 알고리즘 (날씨 코드 기반)
- 한국시간 기준 실시간 영업 상태 (토요일 정기휴무 자동 판별)
- 반응형 디자인 (PC · 태블릿 · 모바일 자동 대응)
- 사용자 후기 + 추첨 이벤트 (Google Sheets 자동 저장)

### 2) 매칭 페이지 — `/match`

학교 학생들 사이의 자연스러운 인연을 만드는 두 가지 방식:

**인스타팅 · 소개팅 신청**
- 익명 신청서 → 운영자가 매칭 → 인스타로 연결
- 카페 결제 시각으로 방문 인증
- MBTI · 이상형 · 학과 · 어필 사항 선택 공개

**수작부리기 게시판**
- 포스트잇 UI로 인상착의 등록
- "저인 거 같아요" 답장 + 날짜/시간 선택
- 작성자만 비밀번호로 답장 열람 가능

### 3) 부속 콘텐츠 자산

- 인스타 카드뉴스 (1080×1080, 10장 × 2종 톤)
- 인쇄용 포스터 (1080×1920, 2종 톤)
- 네이버 블로그 발행글
- 에브리타임 홍보글

## 어떻게 만들었나 · 도구와 워크플로

생각보다 빨리 만들었어요. 도구는 다음 4가지:

- **Claude Code + VS Code** — AI 코페어 프로그래밍 환경. 디자인 톤 · 기능 · 콘텐츠를 프롬프트로 빠르게 반복 수정
- **Netlify** — 폴더만 드래그하면 즉시 배포되는 정적 호스팅 (무료)
- **Open-Meteo API** — 무료 날씨 API (회원가입 / API 키 불필요)
- **Google Apps Script + Sheets** — 백엔드 서버 없이 폼 데이터를 시트에 자동 저장

워크플로는 단순했어요:

1. 프롬프트로 코드 생성 (예: "카페 추천 페이지 만들어줘, Fraunces 세리프 + 크림 컬러")
2. VS Code에서 결과 확인 + 미세 조정
3. Netlify에 폴더 드래그 → 즉시 라이브
4. 모바일에서 확인
5. 마음에 안 드는 부분 다시 프롬프트로 수정

약 15회 이상 반복했어요. AI가 처음부터 완벽한 코드를 주는 게 아니라, "이런 결과를 원해" → "이렇게 바꿔줘" 의 협업에 가까웠어요.

### 매칭 페이지가 가장 어려웠어요

기술적으론 매칭 기능 구현이 제일 어려웠어요. 단순 폼 제출이 아니라:
- 게시글 등록 / 조회
- 답장 등록 / 조회
- 비밀번호 보호 (해싱 처리)
- 익명 카드 그리드 (공개 동의자만)

이 모든 걸 정적 사이트에 백엔드 없이 만들어야 했어요. 결국 Google Apps Script로 라우팅을 직접 짜고, JSONP로 CORS 우회하면서 풀었어요. AI 도움 받았지만 동작 원리는 이해해야 디버깅이 됐어요.

## 가장 인상 깊었던 부분

기술보다 더 의미 있었던 건 **카페 사장님의 가치를 디지털로 옮기는 작업**이었어요.

"10년 직접 로스팅", "시럽까지 수제", "매주 토요일 정기휴무" — 이런 정성과 디테일이 단순한 텍스트 나열로는 잘 전달되지 않아요. 페이지 디자인 톤(Fraunces 세리프 + 손글씨 강조 + 크림 / 페이퍼 컬러)을 카페 분위기에 맞춰 잡는 데 가장 오래 걸렸어요.

또 한 가지 — **"인연"이라는 단어**를 직접 시스템화한 경험이 신기했어요. 매칭 페이지를 만들면서 "수작은 거는 거예요. 인연은 받는 거고요."라는 카피가 나왔는데, 이게 페이지 전체의 톤을 잡아줬어요. 결과적으로 카페 이름인 '수작' 이 단어 하나가 카페 + 매칭 시스템 양쪽을 연결하는 키워드가 됐어요.
