## 🎯 게시 팁 — 후기 모집용

1. **본인 학과 명시 OK** — "글로벌경영학과" 라고 밝히면 같은 학과 학우가 우선 반응. 거리감 줄이는 효과
2. **"보고서 마감" 어필** — 학생들끼리 공감대 강한 키워드, 도움 요청 명분이 자연스러움
3. **추첨 비율 솔직하게** — 묻는 사람 있으면 응모자 N명 중 M명 추첨이라 답변 (당첨 확률 미스리딩 X)
4. **댓글 응대 필수** — "추첨 인원 몇 명?" "마감 언제?" 같은 질문엔 빠르게 답해야 신뢰감 ↑
5. **시간대** — 점심(12~14시) / 저녁(19~22시) — 학생 폰 보는 골든타임
6. **재업로드 금지** — 한 번만, 묻혔으면 톤 다른 버전으로 7일 뒤
7. **카페 광고처럼 보이지 않게** — 본문 전반부에 "비교과 보고서" 키워드 먼저, 카페 정보는 후반

## 💬 예상 질문 + 대응 멘트

**Q. 사장님 부탁받은 거 아님?**
A. 아뇨, 비교과 프로그램 과제로 제가 직접 만든 페이지예요. 사장님은 후기 보시진 않고 보고서용으로 제가 수집하는 거예요.

**Q. 추첨 몇 명?**
A. 응모자 수 보고 결정할 예정인데, 최소 N명은 무조건 뽑을 예정이에요. (← 본인 예산/사장님 합의 따라 숫자 정해두기)

**Q. 페이지가 뭔데?**
A. 조치원 카페수작 메뉴 추천해주는 페이지인데, 날씨랑 기분 골라서 메뉴 추천받는 식이에요. 3분이면 다 둘러봐요.

**Q. 카공할 때 쓰기 좋겠다 / 사장님 어떰?**
A. (자연스럽게 매장 자랑 한 줄 + 다시 후기 부탁)

**Q. 보고서 언제까지?**
A. (실제 마감 솔직하게 — 거짓말하면 신뢰 깨짐)

## 📊 보고서에 어떻게 녹일지 (참고)

수집한 후기 데이터는 "성과보고서 — 사용자 피드백" 슬라이드에 정량 + 정성으로 정리:

**정량 데이터 (Google Sheets에서 함수로 계산)**
- 평균 별점 / 별점 분포 그래프
- 첫인상 분포 (깔끔/따뜻/명확/복잡 등 비율)
- 가장 도움된 기능 TOP 3 (체크박스 카운트)
- 방문 경험 분포 (자주/가끔/한번/없음)

**정성 데이터 (실제 후기 인용)**
- 긍정적 후기 3~5개 발췌
- 개선 의견 3~5개 발췌
- 보고서에 익명 처리 (닉네임 → "A님", "B님" 등)

→ "AS-IS vs TO-BE" 슬라이드에 개선 의견 반영해서 "다음 단계 계획" 으로 자연스럽게 연결
