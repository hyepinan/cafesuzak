각 슬라이드는 정확히 1080×1080px 크기로 제작되어 있어요. 다음 방법으로 깔끔하게 추출하세요:

  1. 크롬에서 F12 (개발자 도구) 열기
  2. 저장하고 싶은 슬라이드 위에서 우클릭 → "Inspect" (검사)
  3. 해당 .slide 요소가 선택됨
  4. 요소 위에서 우클릭 → "Capture node screenshot" 클릭
  5. 1080×1080 PNG 파일이 다운로드 폴더에 자동 저장됨
  6. 10번 반복 → 10장 모두 추출 → 인스타에 슬라이드 게시
